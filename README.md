# Pixel-Leap

this project is for my training of making game in Unity, also my first project in Unity.
the assets used in this project are:

1. "Pixel Adventure 1" by Pixel Frog : https://assetstore.unity.com/packages/2d/characters/pixel-adventure-1-155360
2. "FREE Casual Game SFX Pack" by Dustyroom : https://assetstore.unity.com/packages/audio/sound-fx/free-casual-game-sfx-pack-54116
3. "Casual Game BGM #5" by B.G.M : https://assetstore.unity.com/packages/audio/music/casual-game-bgm-5-135943

and im using a great beginner friendly tutorial on youtube by "Coding in Flow" : https://www.youtube.com/@codinginflow
and the tutorial im using is "Build a 2D Platformer Game in Unity | Unity Beginner Tutorial" : https://youtube.com/playlist?list=PLrnPJCHvNZuCVTz6lvhR81nnaf1a-b67U

go check it out if you want to learn how to make a simple yet amazing game in unity.
